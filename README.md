# CoinWatchBot

Personal Telegram bot that tracks crypto prices and sends private threshold or percentage alerts, supports quiet hours, morning summaries, per-alert cooldowns, and owner usage stats.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
